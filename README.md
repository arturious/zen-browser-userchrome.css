# zen-browser-userchrome.css

Custom userChrome.css for Zen Browser.

## Features
* **URL Bar Centering**: Centers the URL input text and sets font size to `12px`.
* **Clean UI**: Hides the zoom button inside the URL bar.
* **Black Theme**: Matches the URL bar container background to `#000000`.
* **High Contrast**: Sets icons inside the URL bar to white `#ffffff`.
* **Width Tweak**: Limits maximum width of the URL bar to `600px` and centers it on the toolbar.

## Installation
Link or copy `userChrome.css` into your Zen Browser profile's `chrome` directory.
